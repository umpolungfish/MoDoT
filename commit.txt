The return leg imscribed as an ob3ect, and run through itself.

The cycle now takes a single tuple (`imasm cycle tuple=⟨…⟩`), catalog entry
or not, so an object can be put through its own return leg. The return leg
has been imscribed as an ob3ect through the live designer, and reading it
back recovers exactly at all twelve axes.

Two details of that tuple earn their place in the document. Its Ř axis
carries neither member of the colliding pair, so the one place the cycle can
be ambiguous is not where this object sits. Its Φ axis carries one of the
two byte-identical programs, and survives only because position separates it
from its twin on another axis: the object about the ambiguity is readable
only by the mechanism it describes.
