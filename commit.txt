Document the alchemical bench, jam mode, cycles/eagles, and providers

The README covered click and the materials algebra but predated this
session's additions. Brought it current:

  - Quick Start gains the alchemical bench (distill / fdistill / sublime,
    crystallize / cocrystallize / seed, tlc / column, fpt / trap / stain),
    a jam invocation with --cycles / --eagles, and the provider/model flags
    (openrouter default, deepseek + gemini wired).
  - A prose section groups the bench into its four families (volatility,
    crystallization, chromatography, purification/diagnostic) and notes
    each is the same fractionate / Frobenius shape, typed by an ob3ect
    before it was coded.
  - The clarity line is documented: strain and spectral gap read jointly,
    a zero gap is symmetry not a defect, and a two-unit forge is a dimer
    (n >= 3 for a real ring).
  - A jam-mode paragraph: the membrane, the two register notes (a closure
    verifies in its own lane; a non-closure does not deny a proof), and the
    limit that it never checks the mathematics, only refuses to certify what
    the tools did not ground.

    alchemical verbs) and the questions/ description; bumped the date.
