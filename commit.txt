Add the chromatography bench: tlc, column

The third alchemical family, separation by retention, on the axis the
ob3ect typing put both under (retention factor / phase partition), which
is Recognition Ř (index 2):
  - tlc M1 M2...        analytical: spread the set by Rf = mobility
    (1 - norm(Ř), the inverse of retention), group the units into bands,
    and report how many distinct bands appear and which co-elute at the
    same Rf (B). It counts and places, it does not isolate.
  - column M1 M2 ... [on S]  preparative: elute least-retained first with
    the resolution gap to each next fraction (a gap below epsilon is
    co-elution into a shared fraction, B). With `on S`, retention is the
    bond drive to the stationary phase S; without, the intrinsic
    retention norm(Ř). TLC counts, column isolates, exactly the
    analytical-vs-preparative split the oracle typed.

Wired through the Cli flags and the agent verb whitelist as before.
Verified live: TLC resolves a lone high-Rf band from a four-way
co-elution, and column-on-S elutes a non-affine unit first while two
S-binders co-elute at full retention.
