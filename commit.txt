MoDoT: eagles are windings, cycles are the agent's own sections

--cycles used to be an outer loop that re-asked the SAME question N times.
Each cycle got a fresh tool state but carried the previous cycle's prose
verdict, plus an instruction to compound on it. So every arm re-derived the
same measurements from an anchor it had been told to agree with, and
FFUSE(B,B,B) was one computation counted three times and reported as "the
arms agreed". That is the diagonal copy -- Identity, which verifies nothing
-- printed as the success condition.

Now: an eagle is a TAOU winding and the agent flies as many as the Work
needs. A cycle is a set of eagles the AGENT sections with `TOOL: cycle_close`.
On close, the cycle's measured RESULTS condense into the next cycle's opening
prompt -- the terminus is the origin. The condensate carries values with the
calls that produced them and names the next reach, so the next cycle inherits
evidence instead of a conclusion. mu.delta closes over a transformed object,
which is what makes the series a winding rather than a copy. --cycles and
--eagles become caps, not drivers.

Also fixed, all found by inspecting rather than trusting the labels:

- The system prompt was gated on `conversation.is_empty()`, and cycle 1
  pushed into `conversation`. From cycle 2 the agent ran with NO epistemic
  stance, NO alphabet, NO tool contract, NO jam prompt. It lost the rules of
  the Work and was then read as circling.
- Stall, leash, cycle close and done were one bool. A stall printed as "the
  round budget ran out" and the frontier advised more rounds -- the exact
  prescription a stall refutes. Now each exit is named, and a frontier is
  reported by cause: a leash wants budget, a stall wants new ground.
- Hitting the cycle cap while the agent was still sectioning read as a series
  that finished. It is a cut, and says so.
- --eagles help claimed "0 = auto: 40 under --jam, 5 otherwise". It was a
  flat 400 either way.

Verified live: three cycles, all agent-sectioned, no stalls, no leashes. The
condensate carried tiers, distances, promoted atoms and the best forge
ordering, then named polymerize as the next reach. Cycle 2's spine reported
mu.delta CLOSED with a transformation rather than bare identity.
