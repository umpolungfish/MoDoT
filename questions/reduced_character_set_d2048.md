|ν_(0,b)|² = 1/(d+1) for every b≠0 iff its cyclic autocorrelation C_m = Σ_k N_k N_(k+m)
obeys

    C_0 = Σ_k N_k² = 2/(d+1)
    C_m = 1/(d+1)              (m ≠ 0)

|ν_(0,0)|²=1, |ν_(0,b)|²=1/(d+1): C_0 = (1/d)(1 + (d−1)/(d+1)) = 2/(d+1),
C_m = (1/d)(1 − 1/(d+1)) = 1/(d+1). ∎
