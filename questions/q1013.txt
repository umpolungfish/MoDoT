# Conventional Proof: Asymptotics of Triangle-Free Graphs with Given Chromatic Number

## Proposition
Let h₃(k) be the minimum number of vertices n such that there exists a graph G = (V, E) with ω(G) < 3 (triangle-free) and χ(G) = k. We seek to establish the asymptotic behavior of h₃(k) and prove that the ratio of successive terms approaches unity:
lim_k→∞ h₃(k+1) / h₃(k) = 1.

## 1. Encoding and Duality [Φ]: The Probabilistic Lower Bound
To find h₃(k), we consider the independence number α(G). Since χ(G) ≥ |V| / α(G), a graph with χ(G) = k must satisfy n / α(G) ≤ k, or n ≤ k · α(G).
By the probabilistic method (Erdős, 1959), there exist graphs with arbitrarily high girth and chromatic number. For triangle-free graphs, the Ramsey number R(3, t) provides the scaling. R(3, t) is the smallest n such that any triangle-free graph on n vertices has an independent set of size t.
It is known that R(3, t) ≈ t² / log t.
Setting t = α(G), we have n ≈ α² / log α. Since k ≈ n / α, we substitute α ≈ n / k:
n ≈ (n/k)² / log(n/k)
k² ≈ n / log(n/k)
This yields the asymptotic growth:
h₃(k) = Θ(k² log k).

## 2. Inverse Structure [Þ]: The Kim-Ajtai-Komlós-Szemerédi Bound
The precise asymptotic for the maximum independence number in triangle-free graphs is:
α(G) ≥ (1/2 − o(1)) (n/d) log d, where d is the average degree.
For the specific case of h₃(k), the best known bounds (Kim, 1995; Shearer, 1983) establish:
h₃(k) ≈ c · k² log k.
The "inverse" task—finding the smallest n for a fixed k—is equivalent to maximizing χ(G) for a fixed n under the constraint ω(G) < 3.

## 3. Regularity and Concentration [Ç]: The Ratio Limit
To prove lim_k→∞ h₃(k+1) / h₃(k) = 1, we utilize the density of the sequence.
Let f(n) be the maximum chromatic number of a triangle-free graph on n vertices.
From the probabilistic construction, f(n) ≈ √(n / log n).
Note that f(n) is a non-decreasing function of n. Furthermore, f(n+1) ≤ f(n) + 1, because adding a single vertex can increase the chromatic number by at most 1.
Because f(n) → ∞ and the jumps are at most 1, the function f(n) takes on every integer value k for sufficiently large n.
Thus, h₃(k) = min { n : f(n) ≥ k }.
Given the asymptotic h₃(k) ∼ c · k² log k:
h₃(k+1) / h₃(k) ∼ [ (k+1)² log(k+1) ] / [ k² log k ]
Expanding the terms:
(k+1)² / k² = (1 + 1/k)² = 1 + 2/k + 1/k² → 1.
log(k+1) / log k = log(k(1 + 1/k)) / log k = (log k + log(1 + 1/k)) / log k = 1 + log(1 + 1/k)/log k → 1.
Therefore:
lim_k→∞ h₃(k+1) / h₃(k) = 1 · 1 = 1.
