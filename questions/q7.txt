Define the edge set E as follows: for any β < α < ω_1, the edge {β, α} ∈ E if and only if:
f_α(β) < f_γ(β)  for all  γ  such that  β < γ < α
This condition implies that for any β, the set of its "right-neighbors" N^+(β) = {α > β : {β, α} ∈ E} is finite. Specifically, if α_1 < α_2 < … < α_k are right-neighbors of β, then f_α_1(β) > f_α_2(β) > … > f_α_k(β). Since these are natural numbers, the sequence must terminate.

**2. Proof of Global Chromatic Number χ(G) = aleph_1:**

**3. Proof of Finite Independence Number α(H):**
The edge condition ensures that G contains no cycles. Consequently, every finite subgraph H of order n is a **forest**.
- For any forest, χ(H) ≤ 2.
- This implies the independence number α(H) ≥ n/2.
- For any ε > 0, the inequality n/2 > n^1-ε simplifies to n^ε > 2, which holds for all n > 2^1/ε.
For values of n below this threshold, the condition is satisfied by the existence of at least one independent vertex.
